#  Unii Digital Group Assignment

This repository contains multiple projects developed as part of a **technical submission for the Full-Stack Developer position at Unii Digital Group**.

Each project demonstrates different aspects of front-end development, data handling, logic implementation, and responsive layout design.  
All projects are independent and can be run separately.

---

## Repository Structure

```
/
├─ project-one/     # Next.js project (data processing & filtering)
├─ project-two/     # HTML / CSS / JavaScript
├─ project-three/   # HTML / CSS
└─ README.md
```

---

## Project One — Next.js Application

This project focuses on **data aggregation, filtering logic, and table-based data presentation**.

### How to Run Locally

```
cd project-one
npm install
npm run dev
```

Then open:
```
http://localhost:3000
```

### Demo URL

https://unii-digital-group.vercel.app/

---

## Project Two — HTML / CSS / JavaScript

This project is basic front-end logic using **JavaScript** without any framework.

### How to Run

No installation required.

1. Open the `project-two` folder
2. Open `index.html` in your browser
